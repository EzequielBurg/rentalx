# NodeJS Template

A simple NodeJS template with Express, Typescript, eslint, prettier and editorconfig.


## Usage

Clone this template to your machine and run:

```bash
  yarn
```

or

```bash
  npm install
```

to install all dependencies.

## Development
Runs:

```bash
  yarn dev
```
or

```bash
  npm dev
```

to execute the app at port 3333.
